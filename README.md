# java-blackjack
블랙잭 게임 미션 저장소  
  
## 요구사항
### Game
- [ ] 참여자에 따른 게임을 생성한다.
- [ ] 2장씩 카드를 나눈다.
  - [ ] 카드를 한 장 뽑는다.
- [ ] 카드를 더 받을지에 대한 여부를 물어본다.
- [ ] 최종 결과를 계산한다.


### Player
- [ ] 카드를 추가한다.
- [ ] 카드들을 계산한다.

### Dealer
- [ ] 카드를 추가한다.
- [ ] 카드들을 계산한다.


### Deck
- [x] 무작위 카드를 한 장 반환한다.

### Card
- [x] 숫자와 문양에 따른 카드 생성
