# java-blackjack
블랙잭 게임 미션 저장소

## 연료 주입 기능 구현 목록  
- 자동차 객체가 가진 기능을 알고 있는 인터페이스가 존재한다.
- 자동차 객체는 네개의 값을 가지고 있다.
    - 연비
    - 여행하려는 거리
    - 차종의 이름 
    - 주입해야할 연료량
- 이동거리 입력
- 차량 별 필요한 연료량 계산
- 연료량 보고서 출력
