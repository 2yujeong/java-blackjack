## 기능 구현 목록
- 참가자 추상 클래스 구현
    - 플레이어 구현
        - 손에 카드를 가지고 있다
        - 카드를 받는다, 안받는다
            - [ ] y, n 이 아닌 다른 값 입력
            - [ ] 공백 또는 NULL
        - 이름을 가지고 있는다
            - [ ] 공백 또는 NULL
    - 딜러 구현
        - 손에 카드를 가지고 있다
        - 점수 총합이 16 이하일시 카드를 받는다
        - 카드를 플레이어들에게 나눠준다
- 카드덱 구현 Deque
    - 카드 구현 Card 클래스
        - 카드 숫자 Enum
        - 카드 무늬 Enum
    - 
- 블랙잭 구현
    - 플레이어들의 이름을 받는다
        - [ ] 플레이어의 이름 개수가 7개 초과
    - 점수를 계산하여 승자를 판단한다
    - 점수가 21을 초과할 경우 패배시킨다
