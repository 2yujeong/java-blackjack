## 기능 구현 목록

- [ ] 카드 덱을 구성한다. (플레잉 카드 규칙에 의거)
- [x] 각 카드는 점수를 가지고 있다.
- [x] 각 카드는 문양을 가지고 있다.
    
- [ ] 각 플레이어(딜러 포함)은 시작 시 2장의 카드를 받는다.
  - [ ] 딜러는 16이하일 시 무조건 한 장을 더 받는다.
  - [ ] 플레이어는 카드를 추가로 받을지 말지 선택할 수 있다.
    
- [ ] 모든 플레이어가 더 이상 받지 않겠다고 하는 경우 점수를 계산한다.
  - [ ] 각각의 플레이어는 딜러와 승패를 가린다.
